Why Forward Model?
  • Flexibility: -> includes systematics (masks, noise, selection effects)
  • Probe combination: -> joint analysis of multiple datasets with shared physics
  • Model complexity: -> capture nonlinear growth, baryons, galaxy formation
  • Methods: -> Full N-body, approximate method, or Neural net based

Requirements
  1. Map with correct $P(k)$
    (power spectrum / 2-pt statistics)
  2. Map with correct wavelet $\ell_1$-norm
    (at scales of interest)
  3. Fast and accurate.

Takeaways

  • Goal: fast κ emulator matching P(k) + wavelet ℓ₁.
  • Method: Gaussian init → GFB iterations → constrained κ.
  • Constraints: ‖P[κ]-Ptgt‖²  ,  ‖ℓ₁[κ]-ℓ₁tgt‖².
  • Results: realistic HOS, good PDF/skew/kurtosis; fast generation.
  • Cost: minutes per map on CPU; no training set needed.
  • Stocasticity:A working model in place but yet to be tested.

What is a “mildly non-linear” scale?

  • Defined by fluctuation amplitude: linear if \( \sigma(R)\!\ll\!1 \), mildly non-linear if \( \sigma(R)\!\sim\!0.5{-}1 \) (equivalently \( \Delta^2(k)\!\equiv\!k^3P(k)/(2\pi^2)\sim\mathcal{O}(1) \)).
  • For weak lensing with sources at \(z_s\!\approx\!2\), the efficiency kernel \(W(z)\propto (1+z)\,\chi(z)\big[1-\chi(z)/\chi_s\big]\) peaks at \(z_\text{peak}\approx 0.97\) (here: \( \chi(z_\text{peak})\approx 3.23\times10^3\) Mpc).
  • Angular scale \(\theta\) probes a comoving scale near the peak: \[ R \approx \chi\!\left(z_\text{peak}\right)\,\theta \quad(\theta\ \text{in radians}). \] Example: $\theta = 15'$ $\Rightarrow$ $R \approx 14.1\ \mathrm{Mpc}$ at the kernel peak ($z\simeq 0.97$).

What has been done for the PDF covariance — and what this implies for $\ell_{1}$

  • Build a semi-analytic model for the PDF covariance (halo-model / perturbative) and calibrate it against simulations (Sun+2022).
  • Use simulation ensembles to capture non-Gaussian off-diagonals and correct the theory in the tails where sampling noise is largest (D’Amico+2020).
  • Combine theory+simulation in a hybrid covariance to avoid the known failure of Gaussian covariance for 1-pt summaries.
  • Implication for our work: the same steps (theory core + simulation correction) can be applied to build a full covariance model for the wavelet $\ell_{1}$-norm.
